## Описание диагностики

Синтаксическая конструкция **Если...Тогда...ИначеЕсли...** не должна иметь одинаковых условий.

## Примеры

```bsl
Если п = 0 Тогда
    т = 0;
ИначеЕсли п = 1 Тогда
    т = 1;
ИначеЕсли п = 1 Тогда
    т = 2;
Иначе
    т = -1;
КонецЕсли;
```

```bsl
// BSLLS:IfElseDuplicatedCondition-off

```
