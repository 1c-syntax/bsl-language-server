## Описание диагностики

Функция "ТекущаяДата" является устаревшей. Рекомендуется использовать функцию "ТекущаяДатаСеанса".

## Примеры
Неправильно:

```bsl
ДатаОперации = ТекущаяДата();
```


Правильно:

```bsl
ДатаОперации = ТекущаяДатаСеанса();
```
