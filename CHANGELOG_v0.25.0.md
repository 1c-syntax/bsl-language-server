## Новые диагностики

- `LogicalOrInJoinQuerySection` - обнаружение использования логического "ИЛИ" в секции соединения запроса
- `QueryNestedFieldsByDot` - обнаружение использования вложенных полей через точку в запросах
- `ServerCallsInFormEvents` - обнаружение серверных вызовов в событиях формы
- `UnsafeFindByCode` - обнаружение небезопасного использования метода `НайтиПоКоду()`
- `UselessTernaryOperator` - обнаружение бесполезного использования тернарного оператора
- `AbstractMagicValueDiagnostic` - базовый класс для диагностик обнаружения магических значений
