#Использование устаревшего метода "Найти"

Функция "Найти" является устаревшей. Рекомендуется использовать функцию "СтрНайти".

Неправильно:

```BSL

Если Найти(Сотрудник.Имя, "Борис") > 0 Тогда

КонецЕсли; 

```


Правильно:
```BSL

Если СтрНайти(Сотрудник.Имя, "Борис") > 0 Тогда

КонецЕсли; 

```
