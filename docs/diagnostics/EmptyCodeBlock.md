# Пустой блок кода

Пустые блоки являются признаком возможной ошибки:

- Забыли реализовать
- Удалили содержимое

Пустые блоки кода должны быть наполнены либо удалены. 

## Параметры

* `commentAsCode` - `Булево` - Считать комментарий в блоке кодом
