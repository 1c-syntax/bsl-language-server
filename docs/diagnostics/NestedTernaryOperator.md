# Вложенный тернарный оператор

Использование вложенных тернарных операторов затрудняет читаемость кода.

Пример некорректного использования тернарных операторов:
``` 
Результат  =  ?(x % 15 <> 0, ?( x % 5 <> 0, ?( x % 3 <> 0, x, "Fizz"), "Buzz"), "FizzBuzz"); 
```
Возможный вариант реализации:
``` 
Если x % 15 = 0 Тогда
	Результат = "FizzBuzz";
ИначеЕсли x % 3 = 0 Тогда
	Результат = "Fizz";
ИначеЕсли x % 5 = 0 Тогда
	Результат = "Buzz";
Иначе
	Результат = x;
КонецЕсли;
```
