# Повторяющиеся условия в синтаксической конструкции Если...Тогда...ИначеЕсли...

Синтаксическая конструкция **Если...Тогда...ИначеЕсли...** не должна иметь одинаковых условий.

Например:

```bsl
Если п = 0 Тогда
    т = 0;
ИначеЕсли п = 1 Тогда
    т = 1;
ИначеЕсли п = 1 Тогда
    т = 2;
Иначе
    т = -1;
КонецЕсли;
```
