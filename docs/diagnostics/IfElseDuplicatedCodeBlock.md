## Описание диагностики

Синтаксическая конструкция **Если...Тогда...ИначеЕсли...** не должна иметь одинаковых блоков кода.

## Примеры

```bsl
Если п = 0 Тогда
    т = 0;
ИначеЕсли п = 1 Тогда
    т = 1;
ИначеЕсли п = 2 Тогда
    т = 1;
Иначе
    т = -1;
КонецЕсли;
```
