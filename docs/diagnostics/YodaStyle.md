# В операциях сравнения переменная должна быть слева от оператора, значение должно быть справа

Сравнивая значения и переменные помни: переменные слева быть должны (С) Yoda 

Плохо:
```BSL
Если 0 = Сумма Тогда
```

Хорошо:
```BSL
Если Сумма = 0 Тогда
```
