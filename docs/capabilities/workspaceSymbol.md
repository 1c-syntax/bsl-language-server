# Поиск символов по проекту

Быстрый переход к любому методу или объекту во всём проекте по имени.

**Горячие клавиши:** `Ctrl+T`

[← Все возможности](index.md)

## Поиск символа по всему проекту

Пользователь нажимает `Ctrl+T` и набирает `КурсВалюты` для поиска символа по всему проекту. Список мгновенно фильтруется до методов и объектов с этим именем, позволяя сразу перейти к нужному в любом модуле конфигурации.

![workspaceSymbol-01](https://github.com/user-attachments/assets/c4635e17-b209-4fbd-8ef6-dea328c0413e)

## Поиск символов: разбиение по словам (CamelHumps)

В поиске символов (`Ctrl+T`) пользователь набирает только первые буквы слов — `ОбщНазн`. Поиск по CamelHumps сопоставляет их с границами слов и находит `ОбщегоНазначения` и связанные с ним символы, не требуя ввода полного имени.

![workspaceSymbol-02-camelhumps](https://github.com/user-attachments/assets/ad7d9b9c-66ae-43d5-b41a-e49397932ef9)

## Поиск символов: по подстроке и ранжирование

В поиске символов (`Ctrl+T`) пользователь вводит подстроку `значен`. Находятся все символы, содержащие этот фрагмент в любой части имени, а результаты ранжируются по релевантности совпадения.

![workspaceSymbol-03-substring](https://github.com/user-attachments/assets/7a9ef663-761b-4bfe-90b3-19fa373ba03d)

## Поиск символов: нечёткий по аббревиатуре

В палитру `Ctrl+T` вводится аббревиатура `ЗначРекО`. Нечёткий поиск по символам проекта находит методы вроде `ЗначенийРеквизитовОбъекта` — буквы `Знач`/`Рек`/`О` сопоставлены как подпоследовательность и подсвечены.
