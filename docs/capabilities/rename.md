# Переименование

Безопасное переименование символа со всеми его использованиями.

**Горячие клавиши:** `F2`

[← Все возможности](index.md)

Курсор ставится на метод, нажимается `F2`, вводится новое имя `Суммировать` и подтверждается `Enter`. Метод и все его использования переименовываются одновременно.
