# Связанное редактирование

Редактирование объявления локального символа (переменной, параметра) одновременно изменяет все его вхождения в модуле — без вызова переименования.

**Горячие клавиши:** `автоматически`

[← Все возможности](index.md)

## Связанное редактирование вхождений локальной переменной

Курсор ставится внутрь объявления локальной переменной `Счётчик`, и к её имени дописываются символы. Без вызова переименования все вхождения переменной в модуле меняются одновременно по мере набора.

![linkedEditing-01](https://github.com/user-attachments/assets/6b524c10-1ee3-47db-8448-203158a24641)

## Связанное редактирование параметра метода

Курсор ставится внутрь имени параметра метода `Заказ`, и к нему дописываются символы. Без вызова переименования все вхождения параметра в теле метода меняются одновременно по мере набора.
