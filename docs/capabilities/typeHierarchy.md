# Иерархия типов

Для классов OneScript, использующих библиотеку наследования `extends`: дерево супертипов и подтипов по аннотациям `&Расширяет` и `&Реализует`.

**Горячие клавиши:** `Показать иерархию типов (контекстное меню)`

[← Все возможности](index.md)

На классе OneScript, использующем наследование `extends`, вызывается «Show Type Hierarchy». Открывается дерево иерархии типов — супертипы и подтипы класса.
