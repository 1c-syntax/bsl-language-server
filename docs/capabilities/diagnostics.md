# Диагностики

Подсветка ошибок, потенциальных проблем и нарушений стандартов прямо в коде и в панели Проблемы.

**Горячие клавиши:** `Ctrl+Shift+M (панель Проблемы)`

[← Все возможности](index.md)

Пользователь открывает модуль `demo.bsl` и нажимает Ctrl+Shift+M, чтобы открыть панель Проблемы. Проблемные места подсвечены прямо в коде, а панель Проблемы перечисляет найденные диагностики с их описаниями.
