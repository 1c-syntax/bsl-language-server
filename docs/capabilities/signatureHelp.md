# Подсказка по параметрам

При вводе вызова метода показывает список параметров и подсвечивает текущий.

**Горячие клавиши:** `Ctrl+Shift+Space`

[← Все возможности](index.md)

## Подсказка по параметрам метода

Пользователь набирает вызов локальной функции `Сумма = Сложить(` и доходит до открывающей скобки. Поверх кода всплывает панель сигнатуры со списком параметров метода `Сложить`, в которой подсвечен активный первый параметр.

![signatureHelp-01](https://github.com/user-attachments/assets/35fb211e-d2b7-417e-b090-605d1a1d4dab)

## Подсказка по параметрам конструктора (Новый Массив)

Пользователь набирает конструктор `Список = Новый Массив(` и доходит до открывающей скобки. Всплывает панель сигнатуры конструктора `Массив` со списком его параметров и подсветкой активного параметра.

![signatureHelp-02-variadic](https://github.com/user-attachments/assets/44e9ffa6-f435-41d1-8e36-f82dfa8d9078)

## Подсказка по параметрам: необязательный параметр

Пользователь набирает вызов функции `Р = Форматировать(` с необязательным параметром `Префикс = "[]"`. Панель сигнатуры показывает оба параметра, отмечая `Префикс` как необязательный (со значением по умолчанию) и подсвечивая активный параметр.

![signatureHelp-03-optional](https://github.com/user-attachments/assets/8001f71e-a18e-4cb8-8f9b-918ce3a66e08)

## Подсказка по параметрам платформенного метода

Пользователь обращается к платформенному методу `Список.Добавить(` у переменной типа `Массив`. Панель сигнатуры показывает параметры платформенного метода `Добавить` с подсветкой активного параметра.

![signatureHelp-04-platform](https://github.com/user-attachments/assets/e44c3a15-438a-499b-a5a7-c0f85337160d)

## Подсказка во вложенном вызове

Пользователь набирает вложенный вызов `Сообщить(Сложить(10, ` и переходит ко второму аргументу внутренней функции. Панель сигнатуры показывает параметры именно внутреннего вызова `Сложить` и подсвечивает активный второй параметр.

![signatureHelp-05-nested](https://github.com/user-attachments/assets/c11a4158-f92b-4299-993a-64a0b446d1c5)

## Подсказка по цепочке вызовов

Пользователь набирает цепочку вызовов `Строка = Таблица.Скопировать().Найти(` и вызывает подсказку по параметрам. Панель сигнатуры определяет тип результата `Скопировать()` и показывает параметры метода `Найти` с подсветкой активного параметра.

![signatureHelp-06-chained](https://github.com/user-attachments/assets/2e3c980d-1063-4d80-8364-b77ef72f878e)

## Метод с переменным числом аргументов (необязательные параметры)

Пользователь набирает вызов `Позиция = СтрНайти("abc-def", "-", ` и переходит к третьему аргументу метода с переменным числом аргументов. Панель сигнатуры показывает все параметры `СтрНайти`, в том числе необязательные, и подсвечивает активный третий параметр.

![signatureHelp-07-overload](https://github.com/user-attachments/assets/c61449e6-847a-445d-be7e-9447d8a8478b)
