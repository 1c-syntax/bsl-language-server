# Форматирование

Форматирование всего документа, выделенного фрагмента и по мере набора (отступы, регистр ключевых слов).

**Горячие клавиши:** `Shift+Alt+F`

[← Все возможности](index.md)

## Форматирование документа

В файле `messy.bsl` с неотформатированным кодом (без отступов) пользователь выполняет команду `Format Document`. Весь модуль переформатируется: добавляются вложенные отступы для тела процедуры, цикла и условия, а ключевые слова приводятся к единому регистру.

![formatting-01](https://github.com/user-attachments/assets/cb74acf8-e7dd-4d1a-992f-a923708682db)

## Форматирование при вводе (отступы на лету)

Пользователь набирает процедуру с вложенными `Если`/`Иначе` строку за строкой, нажимая `Enter` после каждой. Форматирование при вводе автоматически выставляет отступы на лету, так что каждый новый уровень вложенности сразу получает правильный сдвиг.

![formatting-02-ontype](https://github.com/user-attachments/assets/f555c49a-7b99-4d28-a8ed-42935afe6427)

## Форматирование выделенного диапазона

В `range.bsl` пользователь выделяет несколько строк внутри блока `Если` и выполняет команду `Format Selection`. Переформатируется только выделенный фрагмент (ему добавляются отступы), а остальной код процедуры остаётся нетронутым.

![formatting-03-range](https://github.com/user-attachments/assets/92a6b1c1-1204-42c7-86b2-6e7ff5d48543)
