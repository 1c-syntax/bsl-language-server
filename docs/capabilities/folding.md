# Сворачивание кода

Сворачивание процедур, функций, областей и блоков для удобной навигации.

**Горячие клавиши:** `Ctrl+Shift+[ / ]`

[← Все возможности](index.md)

В файле `folding.bsl` пользователь по очереди ставит курсор на блок комментариев, условие и пакет запросов и нажимает `Ctrl+Shift+[`. Каждый блок сворачивается в одну строку с маркером, оставляя видимыми только заголовки и компактную структуру модуля.
