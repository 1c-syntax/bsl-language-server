# Структура документа

Дерево процедур, функций и областей модуля — в панели Структура и в быстром переходе.

**Горячие клавиши:** `Ctrl+Shift+O`

[← Все возможности](index.md)

## Структура документа (символы)

В открытом `.bsl`-модуле нажимается `Ctrl+Shift+O` (быстрый переход по символам). Появляется дерево процедур, функций и областей модуля для навигации.

![documentSymbol-01](https://github.com/user-attachments/assets/06b8c4e6-bf58-435d-bc10-ada9e3560974)

## Дерево символов в панели Структура (Outline)

Открывается панель Структура (Outline) для текущего `.bsl`-модуля. В ней отображается дерево процедур, функций и областей модуля.

![documentSymbol-02-outline](https://github.com/user-attachments/assets/223f2131-948c-48d7-b314-081ab1e50ca8)

## Устаревший метод зачёркнут в структуре

В панели Структура показан модуль с методом `ПосчитатьИтог`, помеченным как устаревший комментарием `// Устарела.`. Устаревший метод отображается зачёркнутым, в отличие от актуального `РассчитатьСумму`.

![documentSymbol-03-deprecated](https://github.com/user-attachments/assets/6cdf4400-b89f-42a5-987b-70d4a8164506)
