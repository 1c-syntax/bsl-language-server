# Поиск использований

Поиск всех мест использования символа по проекту.

**Горячие клавиши:** `Shift+F12`

[← Все возможности](index.md)

## Поиск использований метода по всей конфигурации

Курсор ставится на объявление экспортного метода общего модуля, и нажимается `Shift+F12` (поиск использований). Открывается список всех вызовов этого метода по всей конфигурации, в разных модулях.

![refs-01-method-cross-module](https://github.com/user-attachments/assets/422fb427-5a6c-465e-b9b7-85a7e080b111)

## Поиск использований локальной переменной (в пределах метода)

Курсор ставится на локальную переменную внутри метода, и нажимается `Shift+F12` (поиск использований). Открывается список вхождений переменной, ограниченный пределами этого метода.

![refs-02-local-variable](https://github.com/user-attachments/assets/4fbc6022-14d3-4d1e-9c0b-9a101a080a96)

## Использования метода модуля менеджера по всей конфигурации

Курсор ставится на метод модуля менеджера, и нажимается `Shift+F12` (поиск использований). Открывается список всех вызовов этого метода по всей конфигурации.

![refs-03-manager-module-method](https://github.com/user-attachments/assets/7b7cd73f-30fb-4542-8d39-a7e1f6ddb887)
