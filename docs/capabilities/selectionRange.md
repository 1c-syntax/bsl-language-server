# Умное выделение

Пошаговое расширение и сужение выделения по синтаксическим границам.

**Горячие клавиши:** `Shift+Alt+→ / ←`

[← Все возможности](index.md)

Курсор стоит внутри подвыражения условия `Если` в `selection.bsl`, и пользователь несколько раз нажимает `Shift+Alt+→`. Выделение шаг за шагом расширяется по синтаксическим границам — от подвыражения к условию, затем к блоку `Если`, телу процедуры и всему файлу.
