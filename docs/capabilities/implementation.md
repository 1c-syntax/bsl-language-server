# Переход к реализациям

Для классов OneScript, использующих библиотеку наследования `extends`: переход от метода интерфейса (`&Интерфейс`) ко всем одноимённым методам реализующих его классов (`&Реализует`).

**Горячие клавиши:** `Ctrl+F12`

[← Все возможности](index.md)

На методе интерфейса класса OneScript (`&Интерфейс`, библиотека `extends`) вызывается «Go to Implementations». Открывается список реализаций — одноимённых методов в классах с аннотацией `&Реализует`.
