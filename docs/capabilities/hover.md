# Всплывающая документация

При наведении на символ показывает сигнатуру, тип и описание из комментариев.

**Горячие клавиши:** `наведение мыши`

[← Все возможности](index.md)

## Всплывающая документация метода

Курсор наводится на вызов метода в коде модуля. Всплывающая подсказка показывает сигнатуру метода и его описание из документирующего комментария.

![hover-01-method](https://github.com/user-attachments/assets/cff69a27-5b22-45a0-abac-1681ba649cd1)

## Всплывающая документация переменной

Курсор наводится на переменную в коде модуля. Всплывающая подсказка показывает имя переменной и выведенный тип её значения.

![hover-02-variable](https://github.com/user-attachments/assets/6582bfce-5c67-4e00-9550-c158763d8a44)

## Всплывающая документация платформенного метода

Курсор наводится на платформенную функцию `СтрДлина` в коде. Всплывающая подсказка показывает её сигнатуру, тип возвращаемого значения и описание из документации платформы.

![hover-03-platform-method](https://github.com/user-attachments/assets/49aea1d4-7b36-4469-b969-f97fac58caf5)

## Всплывающая документация конструктора класса OneScript

Курсор наводится на конструктор класса OneScript в выражении `Новый`. Всплывающая подсказка показывает сигнатуру конструктора и описание класса.

## Всплывающая документация ключевого слова

## Тип переменной из поля структуры

Курсор наводится на переменную `НомерЗаказа`, которой присвоено `Заказ.Номер`. Всплывающая подсказка показывает тип `Строка`, выведенный из описания поля структуры в BSLDoc-комментарии.

![hover-06-struct-field](https://github.com/user-attachments/assets/c9ca2085-c7f2-4f5c-b992-5300f43002a7)

## Вывод типа переменной из выражения

Курсор наводится на переменную `Количество`, которой присвоен результат `Массив.Количество()`. Всплывающая подсказка показывает тип `Число`, выведенный из этого выражения.
