# Variable is assigned to itself (SelfAssign)

|  Type   |        Scope        | Severity |    Activated<br>by default    |    Minutes<br>to fix    |     Tags     |
|:-------:|:-------------------:|:--------:|:-----------------------------:|:-----------------------:|:------------:|

## Description

It is meaningless to assign a variable to itself and usually points to an error.
