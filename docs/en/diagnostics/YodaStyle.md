# In comparison operations, the variable must be to the left of the operator, the value must be to the right

Compare values and variables, remember: the variables on the left should be (ะก) Yoda 

Bad:
```BSL
If 0 = Summ Then
```

Good:
```BSL
If Summ = 0 Then
```
