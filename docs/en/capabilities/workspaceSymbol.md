# Workspace symbols

Quickly jump to any method or object across the whole project by name.

**Shortcut:** `Ctrl+T`

## Workspace symbol search

The user presses `Ctrl+T` and types `–ö—É—Ä—Å–í–∞–ª—é—Ç—ã` to search for a symbol across the whole project. The list instantly filters down to methods and objects with that name, letting you jump straight to the right one in any configuration module.

## Workspace symbol search: CamelHumps

In workspace symbol search (`Ctrl+T`) the user types just the leading letters of each word ‚Äî `–û–±—â–ù–∞–∑–Ω`. CamelHumps matching maps them to word boundaries and finds `–û–±—â–µ–≥–æ–ù–∞–∑–Ω–∞—á–µ–Ω–∏—è` and its related symbols without typing the full name.

![workspaceSymbol-02-camelhumps](https://github.com/user-attachments/assets/ad7d9b9c-66ae-43d5-b41a-e49397932ef9)

## Workspace symbol search: substring and ranking

In workspace symbol search (`Ctrl+T`) the user enters the substring `–∑–Ω–∞—á–µ–Ω`. All symbols containing that fragment anywhere in the name are found, and the results are ranked by match relevance.

![workspaceSymbol-03-substring](https://github.com/user-attachments/assets/7a9ef663-761b-4bfe-90b3-19fa373ba03d)

## Workspace symbol search: fuzzy abbreviation

The abbreviation `–ó–Ω–∞—á–Ý–µ–∫–û` is typed into the `Ctrl+T` palette. Fuzzy workspace-symbol search finds methods like `–ó–Ω–∞—á–µ–Ω–∏–π–Ý–µ–∫–≤–∏–∑–∏—Ç–æ–≤–û–±—ä–µ–∫—Ç–∞` ‚Äî the letters `–ó–Ω–∞—á`/`–Ý–µ–∫`/`–û` are matched as a subsequence and highlighted.
