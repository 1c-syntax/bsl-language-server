# Document symbols / Outline

A tree of the module's procedures, functions and regions — in the Outline view and quick navigation.

**Shortcut:** `Ctrl+Shift+O`

[← All features](index.md)

## Document symbols

In an open `.bsl` module, `Ctrl+Shift+O` (go to symbol) is pressed. A tree of the module's procedures, functions and regions appears for navigation.

![documentSymbol-01](https://github.com/user-attachments/assets/06b8c4e6-bf58-435d-bc10-ada9e3560974)

## Symbol tree in the Outline view

The Outline view is opened for the current `.bsl` module. It shows a tree of the module's procedures, functions and regions.

![documentSymbol-02-outline](https://github.com/user-attachments/assets/223f2131-948c-48d7-b314-081ab1e50ca8)

## Deprecated method struck through in the outline

The Outline view shows a module whose `ПосчитатьИтог` method is marked deprecated by a `// Устарела.` comment. The deprecated method is rendered struck through, unlike the current `РассчитатьСумму`.

![documentSymbol-03-deprecated](https://github.com/user-attachments/assets/6cdf4400-b89f-42a5-987b-70d4a8164506)
