# Code folding

Collapse procedures, functions, regions and blocks for easier navigation.

**Shortcut:** `Ctrl+Shift+[ / ]`

[← All features](index.md)

In `folding.bsl` the user places the cursor on a comment block, a conditional and a query batch in turn and presses `Ctrl+Shift+[`. Each block collapses into a single line with a fold marker, leaving only the headers and a compact module structure visible.
